# Data-Engineering-Applications

Salutes,

Here, you can find an ETL pipeline in the folder pipeline.

This ETL pipeline Extracts NY taxi trip data (Yellow), Transforms and ingests the data with ingest_data.py script to postgreSQL.
The pipeline is containerized with Docker.
Moreover, the data is ingested to GCP Bucket and BigQuery.
As a bonus, This pipeline workflow is also orchestrated with Kestra.


This ETL pipeline is currently under work to be developed.
